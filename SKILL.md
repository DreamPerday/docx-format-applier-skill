# DOCX Format Applier - AI Skill

> 一个让AI助手能够自动修正DOCX文档格式的技能。只需提供格式模板，即可将任意文档的页面设置、样式、页眉页脚等格式自动套用，保持内容完整无损。

## 📖 简介

**DOCX Format Applier** 是一个专为AI编码助手设计的技能文件（SKILL.md）。当你向AI（如Cursor、GitHub Copilot、Trae、Codex等）描述格式修正需求时，该技能会引导AI：

1. 询问你的源文档路径和格式模板
2. 自动读取模板中的页面设置、样式定义、页眉格式
3. 生成C#（OpenXML）或Python代码，动态修正目标文档
4. 保护公式、文本框、脚注、批注等特殊内容
5. 输出符合模板规范的完美文档

无需手动记录字号、边距、行距——一切从模板学习。

## ✨ 核心特性

- **零硬编码** – 所有格式值从模板文档动态读取，不写死任何数字
- **智能匹配** – 通过样式名称或内容特征匹配源文档与模板的样式
- **内容保护** – 自动识别并跳过OMML公式、MathType对象、文本框、脚注、批注
- **毕业论文专项** – 支持多级目录同步、编号缩进适配、关键词格式修正
- **横竖版兼容** – 自动保留各分区的横/竖方向
- **增量修改** – 仅修改用户指定范围的格式，其余保持源文档原状
- **验证输出** – 提供Python验证脚本，对比模板与目标文档的格式差异

## 🔧 依赖环境

| 工具 | 最低版本 | 用途 |

## 🚀 快速开始

### 1. 获取技能文件

将以下文件保存到你的项目目录或AI助手的技能文件夹中：

- `SKILL.md`（本技能定义文件）

### 2. 安装依赖

#### Windows（推荐使用winget）
```powershell
# 安装 .NET SDK 8.0
winget install Microsoft.DotNet.SDK.8

# 安装 Python 3.12
winget install Python.Python.3.12
```

#### macOS
```bash
brew install dotnet-sdk python@3.12
```

#### Linux (Ubuntu/Debian)
```bash
sudo apt install dotnet-sdk-8.0 python3
```

### 3. 验证环境
```bash
dotnet --version    # 应显示 8.x.x
python --version    # 应显示 3.8+
```

### 4. 准备示例模板

准备两个DOCX文件：
- `模板.docx` – 包含你期望的格式（页面设置、样式、页眉）
- `我的文档.docx` – 需要被修正格式的源文档

### 5. 在AI助手中调用

向AI助手发送以下消息：

```
请使用docx-format-applier技能，将"我的文档.docx"的格式修正为"模板.docx"的风格。
源文档路径：C:\docs\我的文档.docx
格式模板路径：C:\docs\模板.docx
```

AI会引导你完成后续步骤，并自动生成修正代码。

## 🧩 配置到AI编码助手

### ✅ Cursor

1. 打开Cursor设置 → **Rules for AI**
2. 点击 **Add Rule**，命名为 `docx-format-applier`
3. 将 [SKILL.md](./SKILL.md) 的全部内容粘贴进去
4. 保存后，在对话中提及“套用格式”或“修正DOCX排版”即可触发

或者直接将 `SKILL.md` 放入项目根目录的 `.cursor/skills/` 文件夹中。

### ✅ GitHub Copilot (Codex)

1. 在项目根目录创建 `.github/copilot-instructions.md`
2. 将以下内容追加到文件末尾：
   ```markdown
   ## DOCX Format Applier Skill
   [粘贴SKILL.md的全部内容]
   ```
3. 或者在VSCode中，按 `Ctrl+Shift+P` → `Copilot: Add Custom Instructions`，然后粘贴技能内容。

### ✅ Trae (字节跳动AI IDE)

1. 打开Trae，进入 **偏好设置 → 技能市场**
2. 选择 **导入本地技能**
3. 上传 `SKILL.md` 文件
4. 启用该技能后，在对话中输入“@docx-format-applier”即可调用

### ✅ Continue (开源AI插件)

1. 在 `~/.continue/config.json` 中添加：
   ```json
   "customCommands": [
     {
       "name": "docx-format-applier",
       "description": "根据模板自动修正DOCX格式",
       "prompt": "[粘贴SKILL.md内容]"
     }
   ]
   ```
2. 重启Continue，在聊天框输入 `/docx-format-applier` 使用。

### ✅ 其他AI助手 (如Claude Desktop, ChatGPT)

- **Claude Desktop**: 在项目描述或会话上下文中，直接粘贴 SKILL.md 的内容，然后说明“请按照这个技能指导我修正DOCX格式”。
- **通用方法**: 将技能内容作为系统提示词或前置指令发送给AI。

## 🛠️ 进阶：直接运行C#程序（不通过AI）

如果你希望脱离AI独立运行，可以从技能中提取核心代码：

```bash

dotnet new console -n FormatFixer

# 2. 添加OpenXML包

dotnet run -- --source 我的文档.docx --template 模板.docx
```

## ⚠️ 注意事项

1. **公式保护**：包含OMML或MathType公式的段落不会被修改字号或行距，避免渲染错位。
2. **文本框内容**：文本框中的文字也会被正确格式化（需单独遍历）。
3. **脚注/尾注**：独立部件中的文字会使用脚注样式（通常字号小于正文）。
4. **修订模式**：如果文档处于修订模式，格式修正不会破坏修订标记。
5. **只读文件**：请确保源文档未被其他程序打开，否则保存会失败。
